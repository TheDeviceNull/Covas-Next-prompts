# Covas-Next-Prompts

A collection of custom prompts for Covas:next, the Elite Dangerous virtual assistant.

## How to Use

1. Download the prompt you want to use
2. In Covas:next, go to the assistant settings
3. In the "Prompt Customization" section, paste the file content
4. Save changes

## Compatibility Notes

These prompts have been tested with Covas:next and are compatible with the AI model used by the application. It's recommended to periodically check compatibility with new Covas:next versions.
